# Automated Report Generator

A professional tool that fetches data from internal databases and generates detailed, visually appealing PDF reports.

## Features
- **Data Aggregation:** Connects to multiple data sources (mocked).
- **Automated Scheduling:** Schedule reports to be generated at specific intervals.
- **Enterprise Dashboard:** View and manage your report archive with ease.
- **Reliable Engine:** High-performance PDF generation logic.

## Setup

### Backend (Node.js)
1. Go to `backend/`
2. Run `npm install`
3. Run `npm start` (Runs on port 5005)

### Frontend (Next.js)
1. Go to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

## Tech Stack
- **Backend:** Node.js, Express
- **Frontend:** Next.js, Tailwind CSS, Lucide React
